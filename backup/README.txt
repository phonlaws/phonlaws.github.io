เวอร์ชัน v2.6
- ปรับการ์ด “งานที่กำลังเปิดอยู่” ให้สว่างขึ้น/ตัดกับพื้นหลังมากขึ้น เพื่อมองเห็นว่าเปิดงานอยู่
- เกินเวลา = การ์ดติดแดงอ่อน

วิธีรัน (Windows)
1) ติดตั้ง Python 3.x
2) แตก ZIP
3) เปิด CMD ในโฟลเดอร์นี้
4) python -m pip install -r requirements.txt
5) python server.py

เปิดเว็บ:
- หน้าเปิดงาน:  http://localhost:8080/
- หน้าเครื่องกลาง: http://localhost:8080/kiosk.html
