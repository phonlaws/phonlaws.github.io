Quality Setpoint Adjuster

กรอกค่า Quality (%). ระบบจะแนะนำว่าควรปรับ Set point ไหม และคำนวณ Set point ใหม่ (ถ้ากรอกค่าปัจจุบัน)
Target: 1.60% Min: 1.55% Max: 1.65%

ข้อมูลเข้า

กฎการปรับ (ตามที่กำหนด):
  • 1.55 ≤ Q ≤ 1.65 → ไม่ต้องปรับ
  • 1.65 < Q ≤ 1.70 → ปรับ 0.03 t/hr (ลด)
  • Q > 1.70 → ปรับ 0.05 t/hr (ลด)
  • 1.50 ≤ Q < 1.55 → ปรับ 0.03 t/hr (เพิ่ม)
  • Q < 1.50 → ปรับ 0.05 t/hr (เพิ่ม)
หมายเหตุ: ระบบคิดตามค่าที่กรอกจริง (ทศนิยมได้) เช่น 1.651 จะเข้าช่วง 1.65<Q≤1.70
